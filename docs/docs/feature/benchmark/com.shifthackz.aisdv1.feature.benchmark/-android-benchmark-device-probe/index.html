internal class AndroidBenchmarkDeviceProbe(val context: Context) : BenchmarkDeviceProbe(source)

Collects Android hardware metadata without touching AI inference runtimes.

All platform calls are best-effort and guarded because vendor builds can hide or partially implement the APIs used for GPU and accelerator discovery.
