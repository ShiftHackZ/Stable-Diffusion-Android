ServerSetupIntent

sealed interface ServerSetupIntent : MviIntent(source)

User events handled by the provider setup wizard.

Source-selection intents update search/filter/sort state, provider-form intents update the active configuration form, and local-model intents drive download/delete confirmation flows.
