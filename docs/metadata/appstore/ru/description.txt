SDAI — открытое приложение для генерации изображений с ИИ для тех, кому нужна свобода выбора провайдера на мобильном устройстве.

Пишите промпты, настраивайте параметры, подключайте image provider, которым уже пользуетесь, и храните результаты в чистой локальной галерее. SDAI сделан для художников, энтузиастов, prompt engineers и разработчиков, которым нужен мобильный workflow вокруг их текущего generation setup.

Для гибкого AI art workflow:
- подключайте self-hosted WebUI servers и поддерживаемые hosted APIs
- используйте SDAI Cloud, если нужен опциональный hosted provider в App Store build
- генерируйте на устройстве через поддерживаемые Silicon Diffusion Core ML и PrismML Bonsai модели
- работайте с positive и negative prompts там, где они поддерживаются
- настраивайте размер, seed, steps, CFG scale, sampler, модель, engine или provider settings, если это доступно у провайдера
- используйте Img2Img там, где выбранный provider поддерживает image input
- просматривайте изображения во встроенной галерее
- открывайте details результата, переиспользуйте prompts, делитесь изображениями и сохраняйте их на устройство
- выбирайте язык, плотность галереи, поведение темы и accent color

SDAI не привязывает вас к одному облаку. Используйте AI provider, который подходит сейчас, сохраняйте мобильный workflow и проходите путь от prompt до готового изображения без потери контроля.

SDAI Cloud опционален. Он использует app tokens для hosted generation, а пополнение доступно через rewarded ads или in-app purchases там, где это поддерживается. Open-source проект остается provider-first, а SDAI Cloud регулируется отдельными Terms of Service, которые показываются при настройке.

Без телеметрии. Open source. Provider-first.

Доступность функций зависит от выбранного AI provider.
