SDAI — відкритий застосунок для генерації зображень зі ШІ для тих, кому потрібна свобода вибору провайдера на мобільному пристрої.

Пишіть промпти, налаштовуйте параметри, підключайте image provider, яким уже користуєтесь, і зберігайте результати в чистій локальній галереї. SDAI створений для художників, ентузіастів, prompt engineers і розробників, яким потрібен мобільний workflow навколо їхнього поточного generation setup.

Для гнучкого AI art workflow:
- підключайте self-hosted WebUI servers і підтримувані hosted APIs
- використовуйте SDAI Cloud, якщо потрібен опціональний hosted provider в App Store build
- генеруйте на пристрої через підтримувані Silicon Diffusion Core ML і PrismML Bonsai моделі
- працюйте з positive і negative prompts там, де вони підтримуються
- налаштовуйте розмір, seed, steps, CFG scale, sampler, модель, engine або provider settings, якщо це доступно у провайдера
- використовуйте Img2Img там, де вибраний provider підтримує image input
- переглядайте зображення у вбудованій галереї
- відкривайте details результату, повторно використовуйте prompts, діліться зображеннями та зберігайте їх на пристрій
- вибирайте мову, щільність галереї, поведінку теми й accent color

SDAI не прив'язує вас до однієї хмари. Використовуйте AI provider, який підходить зараз, зберігайте мобільний workflow і проходьте шлях від prompt до готового зображення без втрати контролю.

SDAI Cloud опціональний. Він використовує app tokens для hosted generation, а поповнення доступне через rewarded ads або in-app purchases там, де це підтримується. Open-source проєкт залишається provider-first, а SDAI Cloud регулюється окремими Terms of Service, які показуються під час налаштування.

Без телеметрії. Open source. Provider-first.

Доступність функцій залежить від вибраного AI provider.
