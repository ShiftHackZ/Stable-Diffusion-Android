SDAI (Stable Diffusion AI) — открытый мобильный клиент генерации изображений с ИИ для Android и iOS. Он дает художникам, энтузиастам и разработчикам один понятный workflow для промптов, настройки генерации, просмотра результатов и переключения между нужными провайдерами.

Выбирайте backend под задачу:
- собственный сервер AUTOMATIC1111 или SwarmUI
- community-воркеры AI Horde
- Hugging Face Inference API
- OpenAI Images API
- Stability AI / DreamStudio API
- локальная диффузия на Android через ONNX, где доступно

SDAI держит под рукой привычные настройки Stable Diffusion: positive и negative prompt там, где они поддерживаются, seed, steps, CFG scale, размер изображения, sampler, выбор модели или engine, LoRA, embeddings, hypernetworks, Img2Img, Inpaint, batch generation и параметры отдельных провайдеров.

Приложение сделано для мобильного workflow и не привязывает вас к одному облачному сервису. Можно использовать свой сервер, подключить API keys для hosted providers или генерировать на устройстве с поддерживаемыми Android local engines. Сгенерированные изображения сохраняются во встроенной галерее с деталями, зумом, sharing, нативным сохранением/экспортом и ZIP export.

Без рекламы. Без телеметрии. Без lock-in к одному провайдеру. SDAI — community-driven free software: код можно изучать, issues можно репортить, переводы можно улучшать, а будущий мобильный workflow Stable Diffusion можно развивать вместе с проектом.

Примечание: этот листинг F-Droid относится к Android FOSS build. iOS использует общий мобильный опыт SDAI для remote generation providers, а часть local diffusion возможностей остается Android-only.
