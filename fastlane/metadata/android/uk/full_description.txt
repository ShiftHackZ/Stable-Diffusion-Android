SDAI (Stable Diffusion AI) — відкритий мобільний клієнт генерації зображень зі ШІ для Android та iOS. Він дає художникам, ентузіастам і розробникам один зрозумілий workflow для промптів, налаштувань генерації, перегляду результатів і перемикання між потрібними провайдерами.

Обирайте backend під задачу:
- власний сервер AUTOMATIC1111 або SwarmUI
- community-воркери AI Horde
- Hugging Face Inference API
- OpenAI Images API
- Stability AI / DreamStudio API
- локальна дифузія на Android через ONNX, де доступно

SDAI тримає під рукою знайомі налаштування Stable Diffusion: positive і negative prompt там, де вони підтримуються, seed, steps, CFG scale, розмір зображення, sampler, вибір моделі або engine, LoRA, embeddings, hypernetworks, Img2Img, Inpaint, batch generation і параметри окремих провайдерів.

Застосунок створений для мобільного workflow і не прив'язує вас до одного хмарного сервісу. Можна використовувати власний сервер, підключити API keys для hosted providers або генерувати на пристрої з підтримуваними Android local engines. Згенеровані зображення зберігаються у вбудованій галереї з деталями, зумом, sharing, нативним збереженням/експортом і ZIP export.

Без реклами. Без телеметрії. Без lock-in до одного провайдера. SDAI — community-driven free software: код можна переглядати, issues можна репортити, переклади можна покращувати, а майбутній мобільний workflow Stable Diffusion можна розвивати разом із проєктом.

Примітка: цей листинг F-Droid стосується Android FOSS build. iOS використовує спільний мобільний досвід SDAI для remote generation providers, а частина local diffusion можливостей залишається Android-only.
